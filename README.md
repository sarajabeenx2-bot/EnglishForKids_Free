# EnglishForKidsFree 🌻

A premium, free English learning platform for children aged 3–10 years.

## Features

- **Learning Garden Adventure** theme with animated sky, clouds, hills, and friendly characters
- **Age-adaptive onboarding** (3–5, 6–8, 9–10 years)
- **No login required** — progress saved locally on device
- Alphabet, Phonics, Stories, Vocabulary, Speaking, Grammar, Spelling
- Interactive games, daily challenges, and reward system
- Coloring Studio with art gallery
- Printable worksheets and certificates
- Parent Zone with progress tracking
- Safe Kids Mode — no advertisements

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19 + TypeScript
- Vite
- React Router
- Framer Motion
- Web Speech API
- LocalStorage for progress
